Scripting webpages

This note records a workflow to fix some problems with the links in the website. It can be used as a tutorial in TeXmacs scripting. I show how to batch-modify pages. Written May 13th 2013.

You can run the scripts in this page, pay attention however that they will modify your filesystem, so be careful.

This document refers to the state of the repository checked out at commit

dad6b1358b44534c0f6b0d3eb2e721cf04f440c1.

The goal is to change certain hyperlinks to use a user-defined macro instead, this will make easier in the future to customize the rendering or the location of the linked files.
