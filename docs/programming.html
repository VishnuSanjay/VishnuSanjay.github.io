GNU TeXmacs

GNU TeXmacs is a free document preparation system with structured editing and high quality typography. It is designed to create beautiful technical documents. I contribute to its development and use it everyday for my lecture notes, research papers, presentations or to take notes in seminars. Note that it is not an interface to the TeX/LaTeX typesetting system, but it can export to and import from it.
